#### An Interactive Web App using R's **shiny** package

Please view the *README-documentation.pdf* file included in this repository.  Github does not (at the time of this writing) support the rendering of LaTeX math equations included in the documentation.  As such, a PDF file is provided to incorporate the relevant equations.  Additional references are provided below.

This documentation relates to the web application deployed on **shinyapps** which may be accessed/used at [https://joshuaadampoirier.shinyapps.io/igrf/](https://joshuaadampoirier.shinyapps.io/igrf/).
